## Die Trefferquote: Was bedeuten 60–70 % wirklich?

60–70 % Trefferquote klingt auf den ersten Blick nach: 30–40 % der Trades laufen schief. Stimmt. Aber das greift zu kurz.

Entscheidend im Swingtrading ist das **Chance-Risiko-Verhältnis (CRV)**. Wenn ein Gewinn-Trade drei- bis viermal so viel einbringt wie ein Verlust-Trade kostet, reichen sogar 50 % Trefferquote, um langfristig im Plus zu sein. Karsten Kagels arbeitet laut eigener Aussage mit einem „sehr guten CRV" — kombiniert mit der angestrebten Trefferquote ergibt das historisch betrachtet ein positives Gesamtergebnis.

**Die Kurzfassung**: Riskiere maximal 1 % des Kapitals pro Signal. Ein Gewinn-Trade mit 3-fachem CRV und zwei Verlust-Trades von je 1 % ergibt unterm Strich noch +1 %. Das ist der mathematische Kern des Ansatzes — und warum Trefferquote alleine nie das entscheidende Kriterium ist.
